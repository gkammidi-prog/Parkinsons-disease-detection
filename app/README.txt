This folder contains the Streamlit Web Application.

File List:
- app.py  → Main Streamlit application file

How to Run the App:
1. Open terminal or command prompt
2. Navigate to this folder: cd app
3. Run this command: streamlit run app.py
4. App opens in browser at: http://localhost:8501

Features:
- Input patient voice measurement data
- Get Parkinsons Disease prediction
- View prediction confidence score
